Build a full-stack Fellowship Entrance Exam and Seat Allocation System.

Tech Stack:

* Frontend: React (Vite)
* Backend: Node.js + Express
* Database: PostgreSQL
* Auth: JWT

---

UI DESIGN:

* White theme
* Orange gradient primary color
* Clean dashboards with sidebar
* Particle.js animated login background
* Logo (/public/logo.png):

  * Login page (center)
  * Dashboard top-right

---

USER ROLES:

1. Super Admin
2. Program Admin
3. Exam Coordinator (unit-level)
4. Doctor (Interview Panel)
5. Student (Candidate)

---

ROLE-BASED ACCESS CONTROL:

* Each role sees ONLY relevant data
* Filter by:

  * role
  * program
  * unit/location

* Set preference order
* Upload documents
* View exams
* Take exams

* Create programs & specialities

* Manage candidates
* Assign exam coordinators
* Assign doctors
* View rankings
* Run allocation
* Override allocation

---

EXAM COORDINATOR:

* Manage candidates for assigned unit

* Monitor exam progress

EXAM SYSTEM:

* MCQ exams

* Auto-submit

QUESTION MANAGEMENT:

* Manual entry UI

PSYCHOMETRIC SYSTEM:

DOCTOR DASHBOARD:
